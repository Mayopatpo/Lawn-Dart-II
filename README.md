Overview

The Lawn Dart II is a simple, somewhat reliable (Depending on printer tolerances, correct assembly, and luck), and fun 26mm model rocket with real time telemetry.

Requirements

Inorder to build and fly the Lawn Dart II you will need these things: A 3d printer with a > or = max build height of 170mm and width/length of 145mm, around 80 grams of filament which can be any type (I recommend PETG because it tends to be stronger, withstand weather and dirt better, not catch on fire as much, and not melt as much)
